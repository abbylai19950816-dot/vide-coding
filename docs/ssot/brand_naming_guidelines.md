# Brand Naming Guidelines

## 目的

本文件記錄未來為本系統命名時必須遵守的準則。命名不只要好聽，也要能被台灣使用者自然理解、能延伸品牌故事，並降低撞名、商標、網域與 App 名稱衝突風險。

## 命名前必做檢查

每次提出新名稱前，都必須先做初步可用性檢查。初查不是正式法律意見，但應先排除明顯不可用或風險過高的名稱。

至少檢查：

- 一般搜尋：確認是否已有同名或高度相似品牌，不限領域。
- 商標初查：台灣智慧財產局、USPTO、WIPO 或其他可查資料來源。
- 網域：優先查 `.com`，再查 `.app`、`.tw`、`.com.tw`。
- GitHub：確認同名 user / org / repo 是否已被使用。
- App 名稱：App Store / Google Play 是否已有同名或高度相似 App。
- SaaS / 課程 / 預約 / 健身 / 身心工作相關領域：若有同名或近似名稱，風險提高。

若只查到部分資料，提案時必須清楚標示「初查限制」與「尚未確認項目」。

## 提案格式

每個候選名稱應包含：

- 名稱
- 唸法，需以台灣使用者容易理解的方式標示
- 命名來源
- 品牌故事
- 與產品定位的關聯
- 夢、尼、瑞三隻貓如何被納入品牌世界觀
- 可用性初查結果
- 風險
- 推薦程度

不要只給名稱清單，也不要只因為名稱尚未撞名就推薦。

## 使用者與語感原則

- 目標使用者包含台灣個人老師、小型教室、身心工作者、瑜珈/皮拉提斯/舞蹈/私人教練等。
- 名稱必須讓英文程度普通的台灣使用者也能大致念出來。
- 避免太拗口、太像拼錯、太像外星字的抽象造字。
- 不一定要加 `Class`，但若不加，需透過副標或品牌故事補足「課程預約管理系統」的用途辨識。
- 不要讓名字只像貓咪品牌，導致使用者看不出這是課程預約或教室管理工具。
- 不要讓名字太像學校、補習班、兒童線上課程平台，除非未來定位改變。

## 貓咪品牌世界觀

使用者希望三隻貓都被公平考慮進品牌，不希望只有某一隻或某兩隻被主品牌使用。

三隻貓：

- 夢：可對應 `Meng`、`Moon`、`Momo`、`Mumu` 等方向，但需注意讀音與品牌成熟度。
- 尼：可對應 `Nini`，不建議只用 `Ny`，避免被理解為 New York。
- 瑞：可對應 `Rai`、`Ray`、`Rey`、`Rae`、`Raye` 等方向。

命名策略建議：

- 主品牌可不直接使用任何一隻貓的名字，避免偏重。
- 三隻貓可作為品牌角色、功能助手或視覺吉祥物。
- 若主品牌使用其中一隻或兩隻貓的元素，必須說明另外的貓如何在品牌架構裡被同等納入。

可延伸角色方向：

- 瑞：課表、預約、改期、取消、名額同步。
- 尼：收費、票券、付款狀態、有效方案。
- 夢：課程紀錄、出缺勤、學員照護、老師備註。

## 品牌故事方向

系統的核心不是只有「預約」，而是協助個人老師與小型教室把日常行政整理好。

品牌故事應圍繞：

- 幫老師整理課表、收費、票券、點名與課程紀錄。
- 讓學生能低摩擦地購課與預約。
- 減少行政混亂，讓老師把時間留給教學。
- 保留溫柔、細膩、陪伴感，但仍要像正式可商業化的管理系統。
- 三隻貓是陪在系統裡分工的小助手，不必把主品牌變成太貓或太可愛。

## 日文轉英文方向

可以考慮日文意象轉英文或羅馬拼音，但需符合：

- 台灣人容易念。
- 不會讓使用者誤以為是日文學習平台或日本限定品牌。
- 能與課程管理、整理、收藏、秩序、陪伴等概念連上。
- 若使用日文詞，例如 `Kura`、`Mori`、`Sora`、`Tsuki`，必須先查是否已有大量品牌或商標使用。

## 決策狀態

目前品牌首選定為 `thetshiu`。

## Current Brand Direction: thetshiu

### 名稱

`thetshiu`

對外可視覺化為：

- `thetshiu`
- `the tshiu`

### 命名來源

`thetshiu` 來自台語 `thè-tshiú / thuè-tshiú`「替手」的概念，意思是可以接替事情的人，也呼應台語 `tshiú`「手」。

使用者接受它被英文使用者或台灣使用者理解成 `the tshiu`。這個誤讀反而可以成為品牌故事的一部分：`the 手`、`那雙手`、老師身邊多出來的一雙手。

### 品牌定位

`thetshiu` 不是一般約課系統，而是給個人工作者、自由教練、家教、瑜珈/皮拉提斯老師與身心工作者的學生紀錄與課務管理系統。

核心價值：

- 把紙本筆記本中的學生紀錄數位化。
- 整理學生狀態、課堂日誌、注意事項與長期進度。
- 同時串起購課方案、剩餘堂數、預約、改期、取消、出缺勤、收費與票券。
- 讓老師在學生變多後仍能快速查閱每位學生的狀態。

### 品牌故事

`thetshiu` 是老師身邊的那雙手。

它來自台語「替手」：能接替事情、幫忙分擔的人。對個人老師來說，許多重要資訊原本散落在紙本筆記、LINE 訊息、收款紀錄與腦中的記憶裡。`thetshiu` 像一雙安靜的手，把每位學生的紀錄、方案、票券、預約、收費與課堂日誌放回清楚的位置，讓老師少一點行政混亂，多一點專注在教學與陪伴。

`the tshiu` 也可以被理解成「那雙手」：不是取代老師，而是在老師需要時，把繁瑣的課務穩穩接住。

### 三隻貓的品牌角色

三隻貓都不直接放進主品牌名稱，避免偏重任何一隻。牠們是 `thetshiu` 這雙手裡的三個分工小助手：

- 瑞：整理預約、課表、改期、取消與名額同步。
- 尼：守好購課方案、剩餘堂數、收費狀態與票券扣補。
- 夢：記錄課堂日誌、學生狀態、注意事項與長期進度。

### 初步可用性狀態

已初查：

- `thetshiu.com`：DNS 查不到紀錄，可能可註冊。
- `thetshiu.app`：DNS 查不到紀錄，可能可註冊。
- `thetshiu.tw`：DNS 查不到紀錄，可能可註冊。
- `the-tshiu.com`：DNS 查不到紀錄，可能可註冊。
- `the-tshiu.app`：DNS 查不到紀錄，可能可註冊。
- GitHub `thetshiu`：查無同名 user/org。
- 一般搜尋：未看到明顯同名 App / SaaS / 商標。

仍需正式確認：

- 台灣智慧財產局商標。
- USPTO。
- WIPO。
- App Store / Google Play 名稱。
- 實際網域 registrar 可購買狀態。

### 命名風險

- `tshiu` 對不熟台羅的人仍需要品牌故事輔助。
- `thetshiu` 可能被讀成英文 `the` + `tshiu`，但目前使用者接受此雙關。
- 若未來正式商業化，需先完成商標與網域鎖定。

已討論但不宜直接採用的方向：

- 太通用且容易撞名：`Flow`、`Book`、`Class`、`Studio` 單獨或常見組合。
- 太抽象難念：`Vezulo`、`Tovumo`、`Yemivo` 類型。
- 太貓導向：`PawClass`、`MewClass`、`NekoClass` 類型，容易讓人看不出是課程預約管理系統。
- 只納入部分貓名：例如只使用瑞與夢的組合，除非另有完整品牌架構讓尼也被公平納入。
